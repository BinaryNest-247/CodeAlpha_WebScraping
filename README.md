# CodeAlpha Web Scraping Project

## Internship Task 1 – Web Scraping

### Objective
To extract book details from a website using Python.

### Tools Used
- Python
- Requests
- BeautifulSoup
- Pandas

### Process
- Sent HTTP request to website
- Parsed HTML using BeautifulSoup
- Extracted book name and price
- Stored data into CSV file

### Output
Generated structured dataset (books.csv) containing scraped book details.
